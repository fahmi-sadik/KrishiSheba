## 🎯 প্রধান বৈশিষ্ট্য

### ব্যবহারকারী ব্যবস্থাপনা
```
প্রশাসক → ব্যবহারকারী যোগ করুন → অনুমোদন করুন → সম্পূর্ণ প্রোফাইল দেখুন
```

### পণ্য অনুমোদন ওয়ার্কফ্লো
```
কৃষক → পণ্য যোগ করুন → অপেক্ষমান অবস্থা → প্রশাসক অনুমোদন → অতিথি/ক্রেতা দেখতে পারে
```

### অর্ডার প্রক্রিয়া
```
ক্রেতা → পণ্য ব্রাউজ করুন → অর্ডার করুন → অপেক্ষমান অবস্থা → কৃষক ডেলিভার করে
```

### রিপোর্টিং
```
প্রশাসক → বিক্রয় প্রতিবেদন → তারিখ ফিল্টার → মাসিক বিশ্লেষণ → প্রিন্ট
```
