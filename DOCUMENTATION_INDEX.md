## 💡 দরকারী টিপস

### দ্রুত সমস্যা সমাধান
```bash
# ক্যাশ সাফ করুন
php artisan cache:clear

# ডাটাবেস রিসেট করুন
php artisan migrate:reset
php artisan migrate
php artisan db:seed

# স্টোরেজ লিঙ্ক পুনরায় তৈরি করুন
php artisan storage:link
```

### ডিবাগিং
```bash
# লগ দেখুন
tail -f storage/logs/laravel.log

# রুট দেখুন
php artisan route:list

# সব কমান্ড দেখুন
php artisan list
```
