# ⚡ KrishiSheba - তাৎক্ষণিক চেকলিস্ট

## 🎯 ৫ মিনিটের মধ্যে চালু করুন

### ✅ ধাপ ১: ডাটাবেস তৈরি করুন (১ মিনিট)

```bash
mysql -u root -p
CREATE DATABASE krishisheba;
EXIT;
```

### ✅ ধাপ ২: Composer ডিপেন্ডেন্সি (২ মিনিট)

```bash
cd c:\xampp\htdocs\KrishiSheba
composer install
```

### ✅ ধাপ ৩: পরিবেশ সেটআপ (১ মিনিট)

```bash
copy .env.example .env
php artisan key:generate
```

### ✅ ধাপ ৪: ডাটাবেস মাইগ্রেশন (১ মিনিট)

```bash
php artisan migrate
php artisan db:seed
```

### ✅ ধাপ ৫: সার্ভার চালু করুন

```bash
php artisan serve
```

**ব্রাউজার খুলুন:** `http://localhost:8000`
