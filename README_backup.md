# KrishiSheba - কৃষি পণ্য ই-কমার্স প্ল্যাটফর্ম

KrishiSheba একটি আধুনিক কৃষি পণ্য বিক্রয় প্ল্যাটফর্ম যেখানে কৃষকরা তাদের পণ্য সরাসরি ক্রেতাদের কাছে বিক্রি করতে পারেন।

## ✨ বৈশিষ্ট্য

### প্রশাসক মডিউল
- ব্যবহারকারী পরিচালনা (অনুমোদন, প্রোফাইল ভিউ)
- পণ্য অনুমোদন ব্যবস্থা
- বিক্রয় প্রতিবেদন এবং বিশ্লেষণ
- মাসিক বিক্রয় সারাংশ
- ব্যবহারকারী সংখ্যা এবং পরিসংখ্যান
- ড্যাশবোর্ড সারসংক্ষেপ

### কৃষক মডিউল
- পণ্য যোগ এবং পরিচালনা
- প্রোফাইল পরিচালনা
- বিক্রয় ট্র্যাকিং
- পণ্য সম্পাদনা এবং মুছে ফেলা
- ড্যাশবোর্ড এবং পরিসংখ্যান

### ক্রেতা মডিউল
- পণ্য ব্রাউজ এবং অনুসন্ধান
- বিভাগ অনুযায়ী ফিল্টারিং
- অর্ডার করা এবং ট্র্যাকিং
- ডেলিভারি ঠিকানা ব্যবস্থাপনা
- অর্ডার ইতিহাস
- বাতিলকরণ সুবিধা

### অতিথি মডিউল
- পণ্য দেখা (বিক্রয় মূল্য অনুমোদনের পর)
- বিভাগ অনুযায় ব্রাউজ করা
- পণ্যের বিস্তারিত দেখা

### বিশেষজ্ঞ মডিউল
- প্ল্যাটফর্ম অ্যাক্সেস (ভবিষ্যতের জন্য প্রস্তুত)

## ⚙️ ইনস্টলেশন

### ১. ডাটাবেস তৈরি করুন

```bash
mysql -u root -p
CREATE DATABASE krishisheba;
EXIT;
```

### ২. পরিবেশ কনফিগার করুন

```bash
cd c:\xampp\htdocs\KrishiSheba
copy .env.example .env
```

`.env` ফাইলে নিম্নলিখিত সম্পাদনা করুন:

```
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=krishisheba
DB_USERNAME=root
DB_PASSWORD=
```

### ৩. Composer ডিপেন্ডেন্সি ইনস্টল করুন

```bash
composer install
```

### ৪. অ্যাপ্লিকেশন কী জেনারেট করুন

```bash
php artisan key:generate
```

### ৫. ডাটাবেস মাইগ্রেশন চালান

```bash
php artisan migrate
```

### ৬. স্টোরেজ লিঙ্ক তৈরি করুন

```bash
php artisan storage:link
```

### ৭. সার্ভার চালান

```bash
php artisan serve
```

তারপর ব্রাউজারে খুলুন: `http://localhost:8000`

## 👥 ডিফল্ট ব্যবহারকারী তৈরি করুন

`database/seeders/DatabaseSeeder.php` ফাইল সম্পাদনা করুন এবং নিম্নলিখিত যোগ করুন:

```php
<?php

namespace Database\Seeders;

use Illuminate\Database\Seeder;
use App\Models\User;
use Illuminate\Support\Facades\Hash;

class DatabaseSeeder extends Seeder
{
    public function run(): void
    {
        // প্রশাসক ব্যবহারকারী তৈরি করুন
        User::create([
            'নাম' => 'প্রশাসক',
            'ইমেইল' => 'admin@krishisheba.com',
            'ফোন' => '01XXXXXXXXX',
            'ঠিকানা' => 'ঢাকা, বাংলাদেশ',
            'ভূমিকা' => 'প্রশাসক',
            'পাসওয়ার্ড' => Hash::make('admin123'),
            'অনুমোদিত' => true,
        ]);

        // নমুনা কৃষক তৈরি করুন
        User::create([
            'নাম' => 'নমুনা কৃষক',
            'ইমেইল' => 'farmer@krishisheba.com',
            'ফোন' => '01XXXXXXXXX',
            'ঠিকানা' => 'রাজশাহী, বাংলাদেশ',
            'ভূমিকা' => 'কৃষক',
            'পাসওয়ার্ড' => Hash::make('farmer123'),
            'অনুমোদিত' => true,
            'ব্যবসার_নাম' => 'সবুজ খামার',
        ]);

        // নমুনা ক্রেতা তৈরি করুন
        User::create([
            'নাম' => 'নমুনা ক্রেতা',
            'ইমেইল' => 'buyer@krishisheba.com',
            'ফোন' => '01XXXXXXXXX',
            'ঠিকানা' => 'চট্টগ্রাম, বাংলাদেশ',
            'ভূমিকা' => 'ক্রেতা',
            'পাসওয়ার্ড' => Hash::make('buyer123'),
            'অনুমোদিত' => true,
        ]);
    }
}
```

Seeder চালান:
```bash
php artisan db:seed
```
